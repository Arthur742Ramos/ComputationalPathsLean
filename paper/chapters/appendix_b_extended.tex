\section*{Part III: Extended Library}

The following tables list the modules added to the formalization beyond
the core development of Parts~I and~II, organised by mathematical domain.
Each module is implemented as a self-contained Lean~4 file in the
\texttt{ComputationalPaths} namespace.  All definitions and theorem
statements use \texttt{Path}-valued witnesses; no \texttt{sorry} or
additional axioms are introduced.
